# Projeto-Sistema-API-LojaDeMateriais
Projeto - Etapa 1 - Sistema e API para Loja de Material de Construção.

## Descrição do Projeto
O Projeto desenvolvido neste repositório utilizará todos os conhecimentos adquiridos no processo trainee de estagiário da empresa para desenvolver habilidades e conhecimentos na criação de APIs RESTful em Java utilizando o framework Spring Boot. 

Nesta aplicação, inicialmente, não contaremos com interface gráfica, sendo seus endpoints consumidos via PostMan para fins de teste e validação de rotas.

Implentaremos o código com boas práticas de desenvolvimento, legibilidade e manutenção do projeto utilizando **Clean Architecture**, com modularização das estruturas com controllers, services, repositories, models e *DTOs*.

### *(English Version)*

# Project-System-API-BuildingMaterialsStore

Project - Stage 1 - System and API for a Building Materials Store.

## Project Description

The project developed in this repository will apply all the knowledge acquired during the company’s trainee internship program to build skills and expertise in creating RESTful APIs in Java using the Spring Boot framework.

In this application, there will initially be no graphical interface; its endpoints will be consumed via Postman for testing and route validation purposes.

The code will be implemented following best development practices, focusing on readability and project maintainability using **Clean Architecture**, with modular structures including controllers, services, repositories, models, and *DTOs*.
